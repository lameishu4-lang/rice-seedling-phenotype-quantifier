# 测试清单

项目名称：水稻秧苗图像表型量化与数据管理软件  
英文名称：Rice Seedling Phenotype Quantifier  
版本号：V1.0.0  
测试目标：验证当前版本所有声明功能均可本地运行  
更新日期：2026-05-23

---

## 一、测试前准备

### 1.1 测试环境

建议测试环境：

1. 操作系统：Windows 10 / Windows 11；
2. Python 版本：Python 3.11；
3. 虚拟环境：`.venv`；
4. 运行方式：命令行运行 `python main.py`；
5. 主要依赖：PySide6、opencv-python、numpy、pandas、openpyxl、python-docx、Pillow。

### 1.2 启动命令

在项目根目录执行：

    .\.venv\Scripts\Activate.ps1
    python main.py

### 1.3 测试数据准备

建议准备：

1. 单张水稻秧苗图像；
2. 包含多张水稻秧苗图像的文件夹；
3. 至少一种 JPG 图像；
4. 至少一种 PNG 图像；
5. 一张不适合分割的图像，用于测试失败提示；
6. 一个空文件夹，用于测试批量分析空文件夹提示。
