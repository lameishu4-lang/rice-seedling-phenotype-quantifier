# 1 简介

## 1.1 编写目的

本文档为《水稻秧苗图像表型量化与数据管理软件 V1.0》的使用说明书。

本文档旨在说明软件的开发背景、功能组成、操作流程、运行环境、数据管理方式、结果导出方式和使用边界。用户可根据本文档完成软件启动、参数设置、单张图像分析、批量图像分析、历史记录管理、结果复核和数据导出等操作。

本文档也可作为软件测试、项目验收、功能核对和软件著作权登记申请过程中的辅助参考材料。

## 1.2 使用对象

本软件的主要使用对象包括：

1. 农业科研人员；
2. 农业院校学生；
3. 农业技术推广人员；
4. 水稻育秧相关技术人员；
5. 作物表型图像分析相关开发人员；
6. 需要对水稻秧苗图像进行批量整理和表型指标记录的用户。

使用者需要具备基本的 Windows 软件操作能力，了解图像文件管理、表格文件导出和基础农学表型概念即可使用本软件。

## 1.3 软件应用范围与业务背景

水稻秧苗素质调查是水稻育秧、移栽和栽培管理过程中的重要环节。传统秧苗表型调查通常依赖人工测量和手工记录，存在工作量大、效率低、主观差异明显、数据整理不便等问题。

本软件面向水稻秧苗图像表型量化场景，基于传统图像处理方法，对水稻秧苗图像中的绿色植株区域进行分割，并在用户设定比例尺的基础上计算株高估算、冠幅估算、投影面积、绿色覆盖率、ExG 叶色指数、Green Ratio 和软件内部长势评分等二维图像表型指标。

软件同时提供本地历史记录管理、批量图像分析、统计摘要、可视化展示、结果解释提示、样本复核和数据导出功能，可辅助用户完成水稻秧苗图像数据的整理、记录和复核。

# 2 产品概述

## 2.1 系统开发工具与技术

本软件采用 Python 语言开发，主要由图像处理模块、界面交互模块、数据管理模块、结果导出模块和复核展示模块组成。

### 2.1.1 Python 开发环境

软件使用 Python 作为主要开发语言。Python 具有良好的跨平台能力和丰富的图像处理、数据处理、界面开发生态，适合快速构建科研辅助工具和桌面端数据分析软件。

本软件建议运行在 Python 3.11 环境下，依赖库通过 `requirements.txt` 安装。

### 2.1.2 OpenCV 与 NumPy 图像处理技术

软件使用 OpenCV 和 NumPy 完成图像读取、颜色空间转换、绿色区域分割、形态学处理、连通域分析、掩膜生成、叠加图生成和像素级指标计算。

当前版本采用传统图像处理方法，不包含深度学习模型训练，不调用外部 AI 推理接口。

主要图像处理内容包括：

1. RGB / HSV 色彩空间转换；
2. HSV 阈值分割；
3. ExG 指数计算；
4. HSV 与 ExG 组合分割；
5. 形态学开闭运算；
6. 连通域面积筛选；
7. 掩膜图生成；
8. 原图与掩膜叠加显示。

### 2.1.3 PySide6 桌面端界面架构

软件界面基于 PySide6 构建。PySide6 可用于构建 Windows 桌面端图形界面，支持按钮、表格、弹窗、滚动区域、图像显示和自绘图表等组件。

本软件包含以下主要界面：

4. 历史记录页；
5. 参数设置页；
6. 批量样本复核弹窗；
7. 历史记录详情弹窗。

### 2.1.4 SQLite 本地数据库

软件使用 SQLite 作为本地数据存储方式，用于保存单图分析记录和批量分析成功记录。

数据库中主要保存：

1. 样本名称；

3. 掩膜图路径；
4. 叠加图路径；
5. 分析时间；
6. 图像尺寸；
7. 比例尺；
8. 分割参数；
9. 表型指标；
10. 备注；
11. 软件版本。

SQLite 数据库保存在本地运行目录中，不需要额外安装数据库服务器。

### 2.1.5 openpyxl 与 python-docx 导出技术

软件使用 pandas 与 openpyxl 实现 Excel / CSV 数据导出，使用 python-docx 实现 Word 图文报告导出。

导出内容包括：

1. 单样本 Word 图文报告；
2. 批量分析 Excel 文件；
3. 批量分析 CSV 文件；
4. 历史记录 Excel 文件；
5. 历史记录 CSV 文件。

Excel 文件包含明细数据工作表和统计摘要工作表。CSV 文件保持为明细数据，便于兼容普通表格工具。

# 3 使用说明

## 3.1 软件启动与主界面概览

### 3.1.1 软件启动

在 Windows 系统中进入项目根目录，启动虚拟环境后执行：

```powershell
python main.py
```

软件启动后进入主界面。

主界面左侧为导航栏，右侧为功能页面。用户可通过左侧导航栏切换不同模块。

### 3.1.2 主界面组成

软件主界面主要包括：

1. 首页；
2. 单张图像分析；
3. 批量分析；
4. 历史记录；
5. 参数设置。

各页面之间互相独立，但共享参数设置和本地历史记录数据库。

## 3.2 首页模块

首页用于展示软件名称、版本信息、功能概述和功能边界说明。

首页主要用于帮助用户快速了解软件定位，包括：

1. 软件面向水稻秧苗图像表型量化；
2. 软件采用传统图像处理方法；
3. 软件支持单图分析和批量分析；
4. 软件支持历史记录管理和数据导出；
5. 软件不提供 AI 诊断、病害判断和生产决策。

## 3.3 参数设置模块

参数设置模块用于配置图像分割参数和默认比例尺。

### 3.3.1 分割方法设置

当前软件支持以下分割方法：

1. HSV；
2. ExG；
3. HSV+ExG。

用户可根据图像背景、光照条件和秧苗颜色情况选择合适的分割方法。

### 3.3.2 HSV 参数设置

HSV 参数包括：

1. H 最小值；
2. S 最小值；
3. V 最小值；
4. H 最大值；
5. S 最大值；
6. V 最大值。

HSV 参数用于从图像中提取绿色区域。若分割结果过少，可适当扩大阈值范围；若背景被误分割，可适当缩小阈值范围。

### 3.3.3 ExG 阈值设置

ExG 是一种常见的超绿指数，可用于增强绿色植被区域。

ExG 阈值越高，分割结果越严格；ExG 阈值越低，分割结果越宽松。

### 3.3.4 最小连通域面积设置

最小连通域面积用于过滤较小噪声区域。

当图像中存在较多绿色噪点时，可适当增大该参数。

### 3.3.5 形态学核大小设置

形态学核大小用于控制掩膜平滑和噪声处理效果。

核大小过小可能无法去除噪声，核大小过大可能导致细小植株区域被削弱。

### 3.3.6 默认比例尺设置

比例尺单位为：

```
cm/pixel
```

比例尺用于将像素单位换算为实际长度单位。株高、冠幅和投影面积等实际单位指标均依赖该参数。

### 3.3.7 参数保存与读取

用户修改参数后可点击保存按钮。保存后的参数会被单图分析页和批量分析页读取。

批量分析页提供“重新读取参数”按钮，用户可手动刷新当前分割参数和默认比例尺。

## 3.4 单张图像分析模块

单张图像分析模块用于完成单个样本图像的分析、指标计算、结果解释、记录保存和报告导出。

### 3.4.1 图像导入与预览

点击“导入图像”按钮，选择待分析的水稻秧苗图像。

支持格式包括：

1. JPG；
2. JPEG；
3. PNG；
4. BMP。

图像导入后，软件会显示原始图像，并显示图像名称、尺寸、路径和当前比例尺。

### 3.4.2 绿色区域分割

导入图像后，点击“执行分割”按钮。

软件根据参数设置中的分割方法和阈值生成：

1. 分割掩膜图；
2. 叠加结果图；
3. 分割区域面积；
4. 外接矩形信息。

如果未检测到有效绿色区域，软件会提示用户调整参数或更换图像。

### 3.4.3 表型指标计算

分割完成后，点击“计算指标”按钮。

软件会根据掩膜图和比例尺计算以下指标：

1. 株高估算；
2. 冠幅估算；
3. 投影面积；
4. 绿色覆盖率；
5. ExG 叶色指数均值；
6. Green Ratio；
7. 外接矩形填充率；
8. 软件内部长势评分。

其中，株高、冠幅和投影面积依赖比例尺设置。

### 3.4.4 单图结果解释提示

指标计算完成后，软件会生成单样本结果解释提示。

解释提示包括：

1. 当前样本内部评分区间；
2. 绿色覆盖率复核提示；
3. 外接矩形填充率复核提示；
4. 比例尺设置提醒；
5. 图像质量、分割效果和实际样本状态复核建议。

该提示仅用于辅助查看和复核，不作为农学诊断、病害判断、品种评价或生产决策依据。

### 3.4.5 保存记录

指标计算完成后，用户可点击“保存记录”。

保存时，软件会将以下内容写入 SQLite 本地数据库：

1. 样本名称；
2. 原始图像路径；
3. 掩膜图路径；
4. 叠加图路径；
5. 分析时间；
6. 图像尺寸；
7. 比例尺；
8. 分割参数；
9. 表型指标；
10. 软件版本。

同一轮分析结果只能保存一次，避免重复写入。

### 3.4.6 Word 报告导出

指标计算完成后，用户可点击“导出报告”。

Word 报告包含：

1. 报告标题；
2. 报告生成时间；
3. 软件版本；
4. 样本名称；
5. 原始图像路径；
6. 分割参数；
7. 原始图像；
8. 分割掩膜图；
9. 叠加结果图；
10. 表型指标表；
11. 单样本结果解释提示；
12. 结果说明。

同一轮分析结果只能导出一次，避免重复生成报告。

## 3.5 批量分析模块

批量分析模块用于对文件夹中的多张水稻秧苗图像进行统一分析。

### 3.5.1 选择文件夹

点击“选择文件夹”按钮，选择包含水稻秧苗图像的文件夹。

软件会自动识别以下图像格式：

1. JPG；
2. JPEG；
3. PNG；
4. BMP。

如果文件夹中没有支持格式的图像，软件会提示未发现图像。

### 3.5.2 读取参数

批量分析页会读取参数设置中保存的分割参数和默认比例尺。

用户也可以在批量分析页临时修改统一比例尺。

点击“重新读取参数”按钮后，软件会刷新当前参数显示。

### 3.5.3 开始批量分析

点击“开始批量分析”按钮后，软件逐张处理文件夹中的图像。

处理过程中显示：

1. 当前处理进度；
2. 当前图像名称；
3. 分析成功或失败状态；
4. 每张图像的指标结果或失败原因。

批量分析完成后，软件会恢复操作按钮状态。

### 3.5.4 批量统计摘要

批量分析完成后，软件生成批量统计摘要。

统计摘要包括：

1. 样本总数；
2. 成功数量；
3. 失败数量；
4. 成功率；
5. 株高均值、最小值、最大值；
6. 冠幅均值、最小值、最大值；
7. 投影面积均值、最小值、最大值；
8. 绿色覆盖率均值、最小值、最大值；
9. 长势评分均值、最小值、最大值。

### 3.5.5 批量结果解释提示

软件会根据批量统计结果和内部评分生成解释提示。

提示内容包括：

1. 本批次分析总数、成功数、失败数和成功率；
2. 最高评分样本序号；
3. 最低评分样本序号；
4. 低评分样本数量；
5. 低评分样本复核建议；
6. 分析失败图像复核建议。

解释提示用于辅助用户定位需要重点查看的样本。

### 3.5.6 批量结果可视化

软件提供轻量级可视化图表，包括：

1. 长势评分柱状图；
2. 株高估算柱状图；
3. 评分等级统计条。

柱状图横轴为样本序号，对应下方结果明细表。图表仅用于批量样本之间的相对比较。

### 3.5.7 查看选中样本

批量分析完成后，用户可在结果明细表中选择一条样本记录，点击“查看选中样本”。

样本复核弹窗显示：

1. 样本序号；
2. 样本名称；
3. 分析状态；
4. 原始图像；
5. 分割掩膜图；
6. 叠加结果图；
7. 表型指标；
8. 单样本复核提示。

该功能用于帮助用户复核低评分样本、异常样本或分析失败样本。

### 3.5.8 保存成功记录

批量分析完成后，点击“保存成功记录”，软件会将成功分析的样本保存至历史记录数据库。

说明：

1. 仅成功分析样本会保存；
2. 失败样本不会写入数据库；
3. 保存时会同步保存掩膜图和叠加图；
4. 同一轮批量分析结果只能保存一次；
5. 保存后的记录可在历史记录页面查看。

### 3.5.9 批量结果导出

批量分析完成后，可导出：

1. Excel 文件；
2. CSV 文件。

Excel 文件包含：

1. 明细数据工作表；
2. 统计摘要工作表。

CSV 文件保持为明细数据。

## 3.6 历史记录模块

历史记录模块用于管理已经保存的单图分析记录和批量成功分析记录。

### 3.6.1 历史记录显示

历史记录表格显示：

1. 序号；
2. 样本名称；
3. 分析时间；
4. 比例尺；
5. 株高；
6. 冠幅；
7. 面积；
8. 覆盖率；
9. 评分；
10. 备注。

表格第一列“序号”为当前查询结果的连续显示序号。数据库内部记录 ID 仅用于数据追踪。

### 3.6.2 历史记录统计摘要

历史记录页面会根据当前查询结果生成统计摘要。

统计摘要包括：

1. 当前显示记录数；
2. 平均株高、最小株高、最大株高；
3. 平均冠幅、最小冠幅、最大冠幅；
4. 平均投影面积、最小投影面积、最大投影面积；
5. 平均绿色覆盖率、最小绿色覆盖率、最大绿色覆盖率；
6. 平均长势评分、最低评分、最高评分。

查询、重置和删除记录后，统计摘要会自动刷新。

### 3.6.3 查询记录

用户可在查询框中输入样本名称或备注关键词，点击“查询”按钮。

### 3.6.4 查看记录详情

选择一条历史记录后，点击“查看记录详情”。

历史记录详情弹窗显示：

1. 显示序号；
2. 内部记录 ID；
3. 样本名称；
4. 分析时间；
5. 原始图像路径；
6. 掩膜图路径；
7. 叠加图路径；
8. 原始图像；
9. 分割掩膜图；
10. 叠加结果图；
11. 表型指标；
12. 备注；

若原始图像、掩膜图或叠加图路径失效，弹窗会提示暂无图像或文件路径失效。

### 3.6.5 编辑备注

选择一条记录后，点击“编辑备注”。

用户可输入或修改备注内容。备注保存后，历史记录表格会刷新。

### 3.6.6 删除记录

选择一条记录后，点击“删除记录”。

删除操作只删除数据库记录，不删除原始图像、掩膜图或叠加结果图文件。

Excel 文件包含明细数据工作表和统计摘要工作表。CSV 文件保持为明细数据。

## 3.7 结果导出与文件管理说明

1. 掩膜图；

3. Word 报告；

6. SQLite 数据库文件；

# 4 注意事项与功能边界

## 4.1 图像质量要求

为获得较稳定的分析结果，建议图像满足以下条件：

1. 图像清晰；
2. 光照相对均匀；
3. 背景尽量简洁；
4. 秧苗区域完整可见；
5. 图像中绿色干扰物较少；
6. 拍摄角度相对固定；
7. 同一批次图像尽量保持相似拍摄条件。

若图像背景复杂、光照过暗、秧苗区域遮挡严重，分割结果可能不稳定。

## 4.2 比例尺设置注意事项

比例尺用于将像素单位换算为厘米单位。

以下指标依赖比例尺：

1. 株高估算；
2. 冠幅估算；
3. 投影面积。

正式记录前，应确认比例尺设置是否符合当前图像采集条件。

若比例尺设置错误，厘米单位指标会出现偏差。

## 4.3 分割参数调整说明

当分割结果不理想时，可尝试调整：

1. HSV 阈值；
2. ExG 阈值；
3. 最小连通域面积；
4. 形态学核大小；
5. 分割方法。

常见情况包括：

1. 秧苗区域提取不足：可适当放宽 HSV 或 ExG 阈值；
2. 背景被误分割：可适当收紧阈值；
3. 噪点较多：可增大最小连通域面积；
4. 掩膜边缘较碎：可调整形态学核大小。

## 4.4 结果解释提示边界

软件中的解释提示基于以下信息生成：

1. 二维图像分割结果；
2. 像素统计结果；
3. 比例尺换算结果；
4. 软件内部评分规则；
5. 批量统计摘要。

解释提示用于辅助用户发现需要复核的样本，并不能替代专业农学判断。

## 4.5 非 AI、非诊断、非生产决策声明

本软件 V1.0.0 版本为本地桌面端图像处理与数据管理工具。

本软件：

1. 不包含深度学习训练；
2. 不包含生成式人工智能；
3. 不调用外部 AI 服务；
4. 不提供病害诊断；
5. 不提供产量预测；
6. 不提供品种评价；
7. 不提供田间生产决策；
8. 不作为农业生产管理的唯一依据。

用户应结合实际样本状态、拍摄条件、人工测量结果和专业经验，对软件输出结果进行综合判断。
