# Rice Seedling Phenotype Quantifier

水稻秧苗图像表型量化与数据管理软件 V1.0

## 一、项目简介

本项目是一款面向水稻育秧图像分析场景的 Windows 桌面端软件，主要用于普通二维图像中的水稻秧苗绿色区域分割、尺度换算、表型指标计算、历史数据管理、批量分析和报告导出。

软件基于传统图像处理方法实现，不依赖深度学习模型训练，不调用云端服务，不使用外部智能接口，不包含生成式人工智能服务，也不实现三维重建或农学诊断结论。

本软件适用于水稻秧苗图像表型数据记录、教学演示、科研辅助整理和批量统计分析。

---

## 二、功能特点

### 1. 单张图像分析

- 支持 JPG、JPEG、PNG、BMP 图像导入；
- 支持图像原图预览；
- 支持 HSV、ExG、HSV+ExG 三种绿色区域分割方式；
- 支持分割掩膜图和叠加结果图显示；
- 支持手动比例尺设置；
- 支持水稻秧苗二维表型指标计算；
- 支持单样本记录保存；
- 支持单样本 Word 图文报告导出。

### 2. 表型指标计算

软件基于二维图像分割结果和比例尺换算，计算以下指标：

- 株高估算；
- 冠幅估算；
- 投影面积；
- 绿色覆盖率；
- ExG 叶色指数均值；
- Green Ratio；
- 外接矩形填充率；
- 软件内部长势评分。

### 3. 历史记录管理

- 支持 SQLite 本地数据库保存分析记录；
- 支持历史记录查询；
- 支持备注编辑；
- 支持记录删除；
- 支持连续显示序号；
- 支持数据库内部记录 ID 追踪；
- 支持历史记录 Excel / CSV 导出。

### 4. 批量分析

- 支持选择图像文件夹；
- 支持批量识别 JPG、JPEG、PNG、BMP 图像；
- 支持批量执行绿色区域分割；
- 支持批量计算表型指标；
- 支持显示成功 / 失败状态；
- 支持显示批量分析进度；
- 支持批量结果 Excel / CSV 导出。

### 5. 参数设置

- 支持设置分割方法；
- 支持设置 HSV 阈值；
- 支持设置 ExG 阈值；
- 支持设置最小连通域面积；
- 支持设置形态学核大小；
- 支持设置默认比例尺；
- 支持参数保存和恢复默认；
- 单图分析和批量分析均可读取保存后的参数。


## 七、使用流程

### 1. 单图分析流程

```text
导入图像
↓
执行分割
↓
计算指标
↓
保存记录 / 导出报告
```

说明：

- 未导入图像前，仅“导入图像”按钮可用；
- 导入图像后，“执行分割”按钮可用；
- 分割完成后，“计算指标”按钮可用；
- 指标计算完成后，“保存记录”和“导出报告”按钮可用；
- 同一轮分析结果只能保存一次；
- 同一轮分析结果只能导出一次报告；
- 重新导入或重新计算后进入新一轮分析流程。

### 2. 批量分析流程

```text
选择文件夹
↓
读取参数
↓
开始批量分析
↓
查看结果表格
↓
导出 Excel / CSV
```

说明：

- 批量分析会读取“参数设置”页面保存的分割参数；
- 批量分析使用统一比例尺；
- 每张图像会显示成功或失败状态；
- 失败图像会给出失败原因；
- 批量结果可以导出为 Excel 或 CSV。

### 3. 历史记录流程

```text
保存单图记录
↓
进入历史记录页面
↓
查询 / 查看 / 编辑备注 / 删除
↓
导出 Excel / CSV
```

说明：

- 历史记录表格第一列为连续显示序号；
- 数据库内部记录 ID 仅用于数据追踪；
- 删除记录后，显示序号会重新排列；
- 删除数据库记录不会删除原始图像文件、掩膜图或叠加图文件。

## 八、指标说明

### 1. 株高估算

株高估算基于分割掩膜区域的外接矩形高度计算：

```text
株高估算值 = 掩膜外接矩形高度像素数 × cm_per_pixel
```

该指标为二维图像中的可见高度估算值，不等同于真实三维植株高度。

### 2. 冠幅估算

冠幅估算基于分割掩膜区域的外接矩形宽度计算：

```text
冠幅估算值 = 掩膜外接矩形宽度像素数 × cm_per_pixel
```

### 3. 投影面积

投影面积基于绿色掩膜像素数量计算：

```text
投影面积 = 掩膜像素数 × cm_per_pixel²
```

### 4. 绿色覆盖率

绿色覆盖率表示绿色掩膜区域在有效图像区域中的占比：

```text
绿色覆盖率 = 绿色掩膜像素数 / 有效图像区域像素数
```

### 5. ExG 叶色指数

ExG 绿色增强指数计算公式为：

```text
ExG = 2G - R - B
```

软件统计掩膜区域内的 ExG 均值作为图像叶色相关指标。

### 6. Green Ratio

Green Ratio 计算公式为：

```text
Green Ratio = G / (R + G + B)
```

软件统计掩膜区域内的 Green Ratio 均值。

### 7. 外接矩形填充率

外接矩形填充率用于描述掩膜区域在外接矩形中的填充程度：

```text
外接矩形填充率 = 掩膜像素数 / 外接矩形面积
```

### 8. 长势评分

长势评分为软件内部综合评分，基于绿色覆盖率、Green Ratio 和外接矩形填充率计算。

该评分仅用于同批次样本之间的相对比较，不作为农学诊断、病害判断或生产决策依据。

## 九、参数说明

软件支持以下参数设置：

| 参数           | 说明                              |
| -------------- | --------------------------------- |
| 分割方法       | 支持 HSV、ExG、HSV+ExG            |
| HSV 下限       | 绿色区域提取的 HSV 下限           |
| HSV 上限       | 绿色区域提取的 HSV 上限           |
| ExG 阈值       | ExG 分割中的二值化阈值            |
| 最小连通域面积 | 用于过滤小噪声区域                |
| 形态学核大小   | 用于开运算、闭运算等掩膜后处理    |
| 默认比例尺     | 默认 cm/pixel，用于像素到厘米换算 |

参数保存后会写入本地配置文件，并被单图分析和批量分析读取。
